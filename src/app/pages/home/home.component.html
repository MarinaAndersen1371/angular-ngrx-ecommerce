Welcome to my

ANGULAR NGRX MongoDB

Application!

This is an Angular-NGRX based Application.

In building of the project I am using "Oopcoders.com" tutorial materials, but I have created completely new backend (using Node express, MongoDB, seeder etc. instead of a fake json-server), Shopping Cart and Place Order procedure. Furthermore I have added some features, for example Register, Ticket, User and User Profile Components, Premium and Franchise Memberships, form validation, responsive navbar etc.
The project is not completed yet. Currently I am working on Manager Area (more complex than Admin area), which will allow, for example, to update customer or order status, to validate test completion, to generate coupon etc.
